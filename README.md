## Сделать 

* Анимация при выборе
* Сервис API для курсов, групп и расписания


## Фичи

* API для замен расписания
* Обьединение общего расписания и замен.

## Сборка

``` bash

npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
